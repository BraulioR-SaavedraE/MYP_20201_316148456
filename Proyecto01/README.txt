Para hacer correr el programa se necesita posicionarse en el directorio "Programa" y ejecutar el comando "python Main.py".
Además de contar con Python instalado, se ocupan los siguientes módulos:
1.- requests
2.- configparser

Se pueden descargar en: https://pypi.org/project/requests/ y https://pypi.org/project/configparser/#files respectivamente.

Por otra parte, las pruebas unitarias también requieren de posicionarse en el directorio "Programa" y ahora introducir el
comando "python Pruebas.py".

Ambos programas requieren de conexión a internet.
