Para ejecutar el programa Filtros, basta con posicionarse en Filtros/Filtros/bin/Debug
que se encuentra entre los directorios de este proyecto y abrir el archivo exe.
Es importante hacer notar que la aplicación se hizo originalmente para ser usada en plataformas Windows. 
Así que la compatibilidad del programa con otros sistemas operativos rebasa el alcance de este diseño.

Las pruebas unitarias del programa se pueden correr abriendo el archivo sln Filtros y usando la opción de ejecutar pruebas.
Dados ciertos conflictos con la apertura de recursos con rutas relativas, para que las pruebas corran correctamente, se necesitan
cambiar las rutas de los recursos en cada uno de los métodos que las requieran.
